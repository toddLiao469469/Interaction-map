# Interaction-map

## 專案簡介
本作品主要是一個互動式的地圖導覽。

數媒系老師發的案子

是研究所課程的期末專題

我主要負責互動裝置的程式設計跟電路設計

作品大致上為一個柱體，頂面上有六個按鈕代表六個景點，按其中一個按鈕

會有相應的燈及聲音。

## 材料
-  6顆  LED
-  6顆  按鈕
-  1個  DFP-mp3-play-mini模組
-  1個  喇叭(1W)
-  1個  74hc595
-  6個  10K 電阻(上拉電阻用)
-  1個  9V電池(Arduino供電用)
-  3個  1.5V電池(DFP-mp3-play-mini用) 不另外供電的話雜訊很大

---

本作品需要注意的有74HC595的資料讀入方向以及播放模組的rxpin及txpin

按照網路上接反而無法用,相反卻可以用。
